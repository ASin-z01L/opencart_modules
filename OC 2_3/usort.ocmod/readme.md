## uSort

### Module Description

The module allows you to individually configure the default sorting for each category. If the same product is assigned
to multiple categories, the extension allows you to set a unique display order for it in each of them independently.


### Configuration Instructions

**Sorting Selection:** Go to the Extensions / Extensions menu, select the "Modules" extension type, and open the uSort
module settings.

The module interface presents a list of all website categories. For each category, you can select
a default sorting that will be applied when navigating to the category in the website catalog.

For individual product
sorting, it is necessary to set the value to "By index (in category)" and save the changes. This will allow the system
to take into account individual product indexes when displaying them on the storefront. The "By index (in category)"
sorting option is also available in the catalog on the front end of the website.

**Setting Product Position:** Go to product editing, "Data" tab, "Sort order in category" field.

For convenience, the field view can be toggled: display "All categories" or "Only those where the product is located."
Specify a numerical index (priority) for a specific category. You can use positive (1, 2, 3) or negative (-1, -2, -3)
values. **Important:** Do not use "0" for the index.

Products with assigned indexes will be displayed at the beginning of the list according to their values. Other products
will be displayed afterward in the order set by default in the system.


### Versions
The module has been tested on clean builds of OpenCart 2.1, 2.3. If the module does not work correctly when installed
on your site, your system files or templates may have already been modified.

### Описание модуля

Модуль позволяет индивидуально настроить сортировку по умолчанию для каждой категории. Если один и тот же товар
привязан к нескольким категориям, расширение позволяет задать ему уникальный порядок отображения в каждой из них
независимо друг от друга.


### Инструкция по настройке

**Выбор сортировки:** Перейдите в меню Дополнения / Расширения, выберите тип расширения «Модули» и откройте настройки
модуля uSort.

В интерфейсе модуля представлен список всех категорий сайта. Для каждой категории можно выбрать сортировку по умолчанию,
которая будет применяться при переходе в категорию, в каталоге сайта.

Для индивидуальной сортировки товаров, необходимо установить значение «По индексу (в категории)» и сохранить изменения.
Это позволит системе учитывать индивидуальные индексы товаров при их выводе на витрине. Пункт сортировки
«По индексу (в категории)» также доступен в каталоге, фронтальной части сайта.

**Настройка позиции товара:**
Перейдите в редактирование товара, вкладка «Данные», поле «Порядок сортировки в категории».

Для удобства вид поля можно переключать: отображать «Все категории» или «Только те, в которых расположен товар».
Укажите числовой индекс (приоритет) для конкретной категории. Можно использовать положительные (1, 2, 3) или
отрицательные (-1, -2, -3) значения. **Важно:** Не используйте «0» для индекса.

Товары с назначенными индексами будут выведены в начале списка согласно их значениям. Остальные товары отобразятся
следом в порядке, установленном в системе по умолчанию.


### Версии

Модуль протестирован на чистых сборках OpenCart 2.1, 2.3. Если при установки на ваш сайт, модуль работает не корректно,
возможно ваши системные файлы или шаблоны уже подвергались изменениям.

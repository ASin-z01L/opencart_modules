## uSort

The module allows you to individually configure the default sorting for each category. If the same product is assigned
to multiple categories, the extension allows you to set a unique display order for it in each of them independently.


## Shortcodes

The module allows inserting standard OpenCart modules (HTML blocks and banners) anywhere within the text description
of products, categories, and articles using shortcodes.


=======================================


## uSort

Модуль позволяет индивидуально настроить сортировку по умолчанию для каждой категории. Если один и тот же товар
привязан к нескольким категориям, расширение позволяет задать ему уникальный порядок отображения в каждой из них
независимо друг от друга.


## Shortcodes

Модуль позволяет вставлять стандартные модули OpenCart (HTML-блоки и баннеры) в любое место текстового описания
товаров, категорий и статей с помощью коротких кодов (шорткодов).
